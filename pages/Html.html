Теги, для оформления любой страницы HTML
  1. <!DOCTYPE html >
  2. < html > lang="ru" оборачивает в себя весь документ
  3. < head > Тег, в котором хранится служебная информация сайта.Может включать следующие теги;
    • < title > bloognoob Заголовок страницы в браузере
    • < link/ > rel="stylesheet" href="css/main.css Добавляет файл CSS в Html
    • < meta/ > charset="utf-8" Кодировка текста в Html
  4. < main > В данный тег оборачивается весь контент и все остальные теги так же
  5. < header > Шапка сайта
  6. < nav > Навигация
  7. < section > Отдельная секция. Как новая глава...
  8. < footer > Подвал сайта
  9. < aside > Сайт-бар сайта
  10. < h1...h6 > Уровни значимости заголовков
  11. < p > Абзац. Параграф
  12. < ul > Неупорядоченный список
  13. < ol > Упорядоченный список
    • reversed Атрибут расставляющий элементы списка в реверсивном порядке
    • type Раскладка написания
  14. < li > Элементы какого-либо списка
  15. < pre > ПЕреформатированный текст. Переносит текст вместе с орфографией в html документ
    16. < code > ОТмечает фрагмент текста, который распознается компьютером
  16. < q > Встроенная цитата
  17. < blockquote > Отдельная цитата. Выделяет текст,как отдельный блог
  18. < a title > Ссылка на ресурс + атрибут заглушка
  19. < cite > Ссыль на источник + ФИО автора и т.д.
  20. < em > Определяет текст, на который сделан акцент
  21. < i > Определяет текст, который отличается от остального, но не является более важным
  22. < strong > Курсив. Указывает на важность текста
  23. < b > Курсив. Нужен для привлечения внимания, но без придания особой важности тексту
  24. < del > Перечеркивание текста
  25. < ins > Подчеркивание текста
  26. < article > Логический раздел, как секция
  27. < dl > Список описаний
  28. < dt > Обозначает термин в списке описаний
  29. < dd > Определение термина в списке описаний
  30. < br > Перенос строки
  31. < sub > Нижний индекс
  32. < sup > Верхний индекс
  33. < time datetime > Дата и время
  34. < div class > Групирование крупных элементов. Работает как контейнер
  35. < span class > Выделяет мелкие текстовые элемены
  36. < ins >
  37. < img src width height > Картинка + путь к картинке, с атрибутами ,задающими ширину и высоту
  38. < figure > Демонстративный материал, которым можно дополнить содержание
  39. < figcaption > Разьясняющий коммент к дополнительному материалу
