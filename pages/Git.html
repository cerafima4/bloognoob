Команды для GIT

CD
Команда в терминале для перехода внужную папку
git pull
Забирает изменения из удаленного репозитория и интегрирует их с изменениями в локальном репозитории
git init
Создание нового репозитория у себя-локально. После ввода команды появляется папка .git
git status
Выводит информацию о том, в каком статусе находится пепозиторий.
git add .
Добавляет в репозиторий все изменения
git commit
Команда делает комментарий тем изменениям, которые были внесены
git push
Используется для выгрузки содержимого локального репозитория в удаленный репозиторий. Она позволяет передать коммиты из локального репозитория в удаленный.
